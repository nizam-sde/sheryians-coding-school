# Responsivness in CSS
## Units

1. px (pixels)  -   fixed units
2. % (percent)  -   depends upon parent 
3. vw (viewport-width)  -   depends upon device width
4. vh (viewport-height)  -   depends upon device height
5. vmax (viewport-maximum)  -   depends upon device width/ height which ever is maximum
6. vmin (viewport-minimum)  -   depends upon width/ height which ever is minimum
7. em (Em - Quadrat)  -   depends upon parent size
8. rem (root em)  -   depends upon root (which is prefixed 16px but can be changed)
