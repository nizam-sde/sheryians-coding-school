# Display Inline & Block

## Display properties

* Block
* Inline        - Inline-block
* none
* flex          - Inline-flex
* grid          - Inline-grid
* Flow-root

## List of elements having block and inline properties list

|      Block    |   Inline   |
|---------------|------------|
|  div          |  a         |
|  p            |  span      |
|  h1  to  h6   |  em        |
|  ul           |  strong    |
|  ol           |  img       |
|  li           |  input     |
|  form         |  label     |
|  header       |  textarea  |
|  nav          |  select    |
|  main         |            |
|  article      |            |
|  section      |            |
|  aside        |            |
|  footer       |            |

        

* Block element take whole width and do not allow any element next to him even there is some empty space
* Inline element take only space till the element itself is, and allow other element to come next to him


## CSS Selectors
